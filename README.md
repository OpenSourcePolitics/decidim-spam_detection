# Decidim::SpamDetection

.

## Usage

SpamDetection will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-spam_detection"
```

And then execute:

```bash
bundle
```
